# algo4_counter_trade — 逆張りカウンタートレード戦略

目的:
- 買いのみ（空売り禁止）
- 当日中にポジション解消（EODフラット）
- サポートライン（価格帯別出来高、日足の山谷・斜め）での反発狙い
- 監視銘柄（相関銘柄）を参考にフィルタ（将来拡張）

開発方針:
- 3分足: 前日・前々日の価格帯別出来高からサポートゾーン抽出（可視化）
- 日足: スイング高安とトレンドライン検出（可視化）
- 過去履歴で売買可能、バックテストで売買位置表示
- 他銘柄にも適用できるようパラメトリックに

## 使い方（概要）
- 入力CSVは jp カラム名（例: 日時, 始値, 高値, 安値, 終値, 出来高）を想定。
- 3分足にリサンプリングし、1日単位でバックテストを実行。
